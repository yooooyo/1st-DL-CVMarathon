## Kernel Density Estimation (KDE)

> 不同 kernel function 的結果

![](https://ai100-fileentity.cupoy.com/ml100/dailytask/1586225294169/1594009345517)

:::info

- 採用無母數方法畫出一個觀察變數的機率密度函數
  - 某個 X 出現的機率為何
  
- Density plot 的特性
  - 歸一：線下面積和為 1
  - 對稱：K(-u) = K(u)
  
- 常用的 Kernel function
  - Gaussian (Normal dist)
  - Cosine
:::
